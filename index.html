How the admin panel works

The sheet is the source of truth. The live leaderboards read directly from the current claim sheet every time the page loads. Anything you do here is an overlay stored in Firestore and re-applied on top of the live numbers.

Override a count. Use “Overrides” below to set a person’s completed-task or redo-counter value for either board. The new value replaces the live one until you remove the overlay.

Add a person not in the sheet. Use “Add overlay-only person”. They’ll show up on the leaderboards with whatever counts you set.

Aliases. If someone is typed two different ways in the sheet (e.g. Antoniio V vs. Antonio V), add an alias to merge them under one display name.

Revert. Editing an overlaid field back to blank reverts that field to the live value.

Start a new week. The “Start new week / swap sheet” button snapshots the current standings (with overlays applied) into the archive, then points the site at a new sheet URL and clears overlays so the new week starts fresh.

Your name. Every adjustment is logged with the name you set above — double-check it before saving.
